    Create a document OCR processor

    Use the Google Cloud console to create a document OCR processor. See creating and managing processors for more information.

    1. In the Google Cloud console navigation menu, click Document AI and select Processor Gallery.

      You can find where it is by clicking the following button:

    2. In the Processor Gallery, search for Document OCR and select Create.

    3. Enter a Processor name, such as quickstart-document-ocr.

    4. Select the region closest to you.

    5. Click the Create button.

    Your processor has now been created.

    Test Processor

    You should now be on the Processor Details page of the processor you just created.

    1. Download the sample document

      It is a PDF file containing the first 3 pages of the classic novel "Winnie the Pooh" by A.A. Milne, which is in the Public Domain in the United States. This file was scanned and digitized by Google Books.

    2. Click the Upload Test Document button and select the document you just downloaded.

    3. You should now be on the Document OCR analysis page. You can view the OCR detected text and document annotations.
